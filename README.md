# 🌙 Sailor Moon Puzzle Game

<p align="center">
  Um jogo inspirado em Candy Crush e Puyo Puyo, ambientado no universo de Sailor Moon.
</p>

## 🎮 Jogar Online
👉 [Clique aqui para jogar](https://sailorgembattle.netlify.app/)


## ✨ Sobre o projeto
Este projeto é um puzzle game desenvolvido com inspiração em clássicos como **Candy Crush** e **Puyo Puyo**, mas com temática de **Sailor Moon**, trazendo uma estética mágica, nostálgica e divertida.

Combina mecânicas de combinação de peças com visual inspirado em animes clássicos, com personagens, efeitos e identidade visual criados com apoio de IA.

## 🚀 Tecnologias e Ferramentas
Projeto desenvolvido com apoio de:

- ChatGPT — lógica, programação e prototipagem
- Claude.ai — refinamento de ideias e suporte de desenvolvimento
- ComfyUI — geração de artes e assets visuais

Tecnologias usadas:
- HTML5  
- CSS3  
- JavaScript

## 🎮 Recursos
- Gameplay estilo match/puzzle  
- Inspiração em Candy Crush / Puyo Puyo  
- Visual temático Sailor Moon  
- Efeitos e arte com auxílio de IA  
- Projeto experimental e autoral
